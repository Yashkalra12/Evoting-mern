Welcome to the eVoting Backend

This backend is built with Node.js and serves as the core of the eVoting System project.

Technologies Used:

  • MongoDB: NoSQL database for data storage
  • Express.js: Backend framework for building RESTful APIs
  • React.js: Frontend library for building dynamic UIs
  • Node.js: Server-side JavaScript runtime

Visit the following routes for different functionalities:

  • POST /api/auth/createuser: Create a new user
  • POST /api/auth/login: User login
  • POST /api/auth/createadmin: Create a new admin
  • POST /api/auth/admin-login: Admin login
